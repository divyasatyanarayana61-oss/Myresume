# Sri Lakshmi Ganapathico - Portfolio & E-commerce Website

This is a premium, elegant, mobile-responsive website for the clothing brand "Sri Lakshmi Ganapathico". The design reflects luxury Indian ethnic aesthetics with a modern touch, using a color palette of maroon, gold, ivory, and beige.

## Features

- **Header & Navigation**: Includes logo, menu items, and cart icon.
- **Hamburger Menu**: Mobile-friendly sidebar with product categories.
- **Hero Section**: Prominent logo display, tagline, and call-to-action button.
- **Responsive Design**: Optimized for mobile and desktop.
- **Soft Animations**: Fade-in effects and hover transitions for a clean UI.

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Media Queries)
- JavaScript (for hamburger menu functionality)

## Setup Instructions

1. Place your brand logo image in the `images/` folder as `logo.png`.
2. Open `index.html` in a web browser to view the website.
3. For a live server, use a local development server (e.g., VS Code Live Server extension).

## Customization

- Update the logo path in `index.html` and `styles.css` if needed.
- Modify colors in `styles.css` to match your exact brand palette.
- Add more sections or functionality as required for the full e-commerce site.
